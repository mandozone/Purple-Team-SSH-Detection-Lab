# Purple Team SSH Brute-Force Detection Lab — Site

Static, single-page presentation site for the lab. Plain HTML + CSS, no build step.

## Deploy

Serve the `site/` directory with any static host:

```bash
# Local preview
cd site && python3 -m http.server 8080
# → open http://localhost:8080

# Or deploy the folder to GitHub Pages, Netlify, Vercel, S3, Cloudflare Pages, etc.
```

## Files

- `index.html` — full presentation page
- `styles.css` — purple/red cybersecurity theme
- `assets/fig1-…fig11-*.jpg` — 11 figures extracted from the lab PDF
- `assets/purple-team-ssh-detection-lab.pdf` — downloadable full lab report
